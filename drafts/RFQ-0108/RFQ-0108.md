# RFQ-0108 — Ecosystem Costing Model

## 1. Overview
Unified cost abstraction system across estate, digital infrastructure, suppliers, and operational systems.

Cost is treated as an event-sourced primitive, not a static attribute.

## 2. Core Model
Cost = time-indexed event stream bound to asset graph nodes.

## 3. Cost Event Schema
- asset_id
- cost_type (CAPEX/OPEX/Lifecycle/Contingency)
- amount
- currency
- timestamp
- supplier_id
- dependency_edges
- confidence_score

## 4. Architecture
- Ingestion (RFQ-0107)
- Normalisation layer
- Graph cost propagation engine
- Evaluation & aggregation engine

## 5. CBS Model
Transforms traditional hierarchical CBS into a DAG enabling:
- multi-parent cost allocation
- shared infrastructure attribution
- dynamic reweighting

## 6. Outputs
- full ecosystem cost graph
- lifecycle cost curves
- supplier reliability-weighted cost exposure

## 7. Constraints
- append-only cost events
- no static cost tables
- all adjustments must be event-sourced
