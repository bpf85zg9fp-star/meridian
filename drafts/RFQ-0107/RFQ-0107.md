# RFQ-0107 — Automated Asset Ingestion Framework

## Overview
Defines deterministic ingestion pipeline for heterogeneous assets across estate, financial, supplier, and temporal domains.

## Pipeline
1. Capture
2. Normalise
3. Classify
4. Dependency resolution
5. Scoring
6. Event store persistence

## Constraints
- Event-sourced only
- No direct mutation of canonical assets
- Unclassified assets prohibited

## Outputs
- A1 asset record
- Graph node
- Event log entry
