# RFQ-1006-001 — Estate Land: Site Selection & Forest Range Lot

## Selected Site

**Lot 100, 10 Fry Lane, Forest Range, SA**

| Parameter | Detail |
|---|---|
| Size | ~13.9 ha (34 acres) |
| Elevation | 252–615 m ASL (average ~465 m) |
| Topography | Rolling hills and elevated plateau; slopes suitable for earth-bermed structures |
| Zoning | Productive Rural Landscape (93%), Conservation (6%) |
| Bushfire rating | BAL-FZ (Flame Zone) |
| Estimated site value | AUD $2,500,000–$3,500,000 |
| Distance to Adelaide CBD | ~35 km / ~45–55 min |
| Distance to hotel site (Hahndorf) | ~22 km / ~22–25 min |
| NBN | FTTP-ready |
| Starlink coverage | Confirmed |

## Why Forest Range

- Elevation provides Adelaide Hills’ characteristic cool climate, views, and privacy
- 34 acres gives adequate APZ (50 m minimum), estate grounds, and bush zones
- Plateau topography enables single-level luxury slab without significant cut-and-fill
- Close enough to Adelaide CBD for daily access; far enough for privacy and rural amenity
- Strong alignment with Adelaide Hills wine tourism for hotel synergy
