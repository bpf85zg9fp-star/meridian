# RFQ-1000-STR-001 — Cladding, Materials & Architectural Palette

## Architectural Intent

Dark, monolithic, refined. The estate reads as a single cohesive object from all approach angles. PGH Blue Steel Flash Metallic brick forms the primary mass; VMZinc Anthra Zinc wraps every non-brick surface — roof, soffits, fascias, garage surround, pavilion cladding, and interior feature reveals. The combination produces a near-seamless dark tonal unity with natural material warmth at close range.

---

## 1. Primary Brick

| Parameter | Specification |
|---|---|
| Brick | PGH Blue Steel Flash Metallic |
| Bond | Double brick construction (120 mm inner leaf + 120 mm outer leaf with insulated cavity) |
| Application | All external walls of main residence and VIP guest suite |
| BAL-FZ compliance | Double brick construction inherently compliant as non-combustible masonry |
| Interior accent | Selected interior feature walls in PGH Blue Steel Flash Metallic (entry gallery, lounge fireplace wall) |

## 2. VMZinc Anthra Zinc

Used on all non-brick external surfaces to create a unified dark-metal palette:

| Application | Detail |
|---|---|
| Roof | Standing seam panels; BAL-FZ rated; long-run (min. seam exposure to ember attack) |
| Soffits and fascias | Concealed fixing; flush with brick reveals |
| Garage door surrounds and entry frames | Wraps Silvelox door openings |
| Outdoor pavilion cladding | Full VMZinc Anthra Zinc facade to match main residence |
| Feature wall reveals | Theatre entry reveal; entry gallery reveals; stair balustrades |
| Theatre entry doors (outer + inner) | VMZinc Anthra Zinc finish Silvelox Secur Plus doors |
| Interior accent panels | Selected feature panels in key living zones |

## 3. Roof

- VMZinc Anthra Zinc standing seam
- BAL-FZ rated (non-combustible)
- Ember-proof gutter and valley mesh (Embertec)
- Internal drainage where possible to eliminate exposed gutter runs
- Motorised skylights in internal courtyard/lightwell zone (not in BAL-FZ exposure zone)

## 4. Glazing

| Type | Specification | Application |
|---|---|---|
| Standard glazing | Triple-glazed, BAL-FZ rated, thermally broken aluminium or steel frames, neutral Low-E tint | Primary living zones, kitchen, lounge |
| Electrochromic smart glass | SageGlass or Halio; instant blackout/privacy; tint-on-demand | All bedrooms and home theatre |
| Frames | Schüco / Reynaers / AWS — thermally broken with fire-rated seals throughout | All glazed openings |
| Standard cost | $1,500–$2,000/m² | |
| Electrochromic cost | $2,500–$3,000/m² | |

## 5. Flooring Palette

| Zone | Flooring |
|---|---|
| Kitchen, bathrooms, mudroom, outdoor paths | Porcelain or natural stone |
| Living and games areas | Karndean Korlok Select or wool carpet |
| Garage and walkways | Polished concrete |
| Formal lawns | Sir Walter Buffalo turf (robot mower-compatible circular design) |
| Hydronic underfloor | Throughout all internal zones |
