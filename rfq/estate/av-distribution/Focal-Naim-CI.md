# RFQ-1000-AV-001 — Focal & Naim CI Whole-Home Audio

## Overview

The Meridian Estate whole-home audio system uses Focal Utopia CI series in-wall/ceiling speakers driven by Naim CI network amplifiers throughout all primary living zones, with Amina invisible speakers in design-sensitive spaces. Roon Nucleus+ provides the music server and streaming integration.

---

## 1. Loudspeakers

| Model | Application | Zones |
|---|---|---|
| Focal Utopia 1000 CI in-wall | Primary living zones: lounge, dining, kitchen, office, hallways | All primary zones |
| Focal Utopia 1000 CI in-ceiling | Bedrooms and zones requiring ceiling install | Bedrooms, library |
| Focal Littora outdoor | Outdoor zones: pavilion, pool area, garden paths | Outdoor |
| Focal Littora underwater | Pool in-water acoustic zone | Pool |
| Amina Mobius invisible | Design-sensitive spaces: bathrooms, dressing suites, formal areas | Bathrooms, dressing |

## 2. Amplification (Naim CI)

| Model | Function | Zones |
|---|---|---|
| Naim CI-Uniti 102 | Network streamer + integrated amp per zone; Apple Music, Qobuz, TIDAL, Roon | Primary zones |
| Naim CI-NAP 108 | 8-channel power amplifier for multi-speaker zones | Open plan, theatre support |
| Naim CI-NAP 101 | Mono amplifier for subwoofers and outdoor power | Outdoor, subwoofer zones |

## 3. Music Server

- **Roon Nucleus+:** Central music library server; integrates Apple Music (via ALAC), Qobuz, TIDAL lossless
- **Apple Music 24-bit/192kHz:** See `rfq/estate/av-distribution/Hi-Res-Audio.md` for dedicated Apple Music hi-res pathway
- **Zone grouping:** Roon zones can be grouped for party mode; individual zone control via Basalte Lena or Josh.ai
