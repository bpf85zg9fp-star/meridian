# RFQ-1000-HT-009 — Mozaex Digital Marquees

## Overview

Two Mozaex Poster Digital Marquee 49" units — one outside the theatre entry, one inside — display Kaleidescape Now Playing artwork, custom artwork, event listings, or programme graphics. Both are integrated with the acoustic millwork of the theatre entry corridor.

## Specification

| Parameter | Specification |
|---|---|
| Model | Mozaex Poster Digital Marquee 49" |
| Quantity | 2 (one exterior entry, one interior lobby) |
| Display | 49" portrait-orientation commercial display |
| Surround | Dark acoustic millwork custom millwork surround |
| Backlighting | RGBW LED backlighting — colour-tuneable via KNX |
| Integration | Kaleidescape API — auto-updates Now Playing poster art |
| Mounting | Recessed flush into acoustic wall panels |

## Kaleidescape Integration

The Mozaex marquees connect directly to the Kaleidescape API. When a movie is selected on the Kaleidescape Strato V:
- Movie poster art automatically loads on both marquees
- Backdrop and metadata displayed in portrait format
- Transitions smoothly between selection and playback states

This creates a true private cinema arrival experience.

## KNX / Josh.ai Integration

- RGBW LED surrounds linked to KNX theatre scenes
- "Pre-show" scene: warm amber glow, poster art displayed
- "Movie" scene: backlights dim to off, display dims to 20%
- "Intermission" scene: backlights pulse softly, full brightness returns
