# RFQ-1000-HT-012 — Barco Heimdall+ (Evaluated, Not Selected)

## Status

This document exists for completeness of the source record. **Barco Heimdall+ was evaluated against Christie Eclipse G3 in the source conversation and was not chosen.** Kevin's own follow-up questions in the same conversation continued exclusively down the Eclipse path ("is a Kaleidescape fed Christie Eclipse with a madVR the pinnacle?"), and the assistant's later design work explicitly states Eclipse was "already selected." This is a distinct product from Barco Thor+ (RFQ-1000-HT-013) — Heimdall+ is Barco's more accessible single-chip residential line, not their flagship.

---

## 1. Specification (As Evaluated)

| Parameter | Barco Heimdall+ | Barco Heimdall+ Cinemascope |
|---|---|---|
| Architecture | Single-chip 4K DLP | Same engine, cinemascope format |
| Resolution | 4,096 × 2,160 (Heimdall CS variant up to 6,144 × 2,592) | — |
| Light engine | RGB laser (discrete red/green/blue) | Same |
| Brightness | 6,000 ANSI lumens | 4,500 ANSI lumens |
| Laser lifetime | ~25,000 hours at full power | Same |
| Dynamic contrast | 20,000:1 via DynaBlack (frame-by-frame laser pulsing + processing) | Same |
| Colour gamut | ~98% Rec. 2020 | Same |
| Connectivity | HDMI 2.1 (HDCP 2.3) | Same |
| Lens options | Multiple, for throw-distance flexibility | Same |
| Control | Embedded geometry correction (warp/blend), network control | Same |

## 2. Comparison Basis (Against Christie Eclipse G3)

| Criterion | Christie Eclipse G3 | Barco Heimdall+ |
|---|---|---|
| Contrast | Up to 20,000,000:1 sequential (6-DLP architecture) | 20,000:1 dynamic (DynaBlack, single-chip) |
| Brightness | 30,000 ANSI lumens, sustained 50,000 hours | 6,000 ANSI lumens |
| Complexity | Requires external chiller, larger footprint, specialist commissioning | Simpler install, more accessible cost/complexity |
| Positioning in source discussion | "Pinnacle" for absolute image fidelity in a dedicated, no-compromise room | "Extremely compelling" where very high colour fidelity is wanted without Eclipse's infrastructure demands |

## 3. Why Not Selected

The source conversation frames Heimdall+ as the right choice for high-end home cinemas that don't have the room, budget, or infrastructure for Eclipse-class equipment. Given Meridian Estate's zero-compromise brief and the theatre infrastructure already committed (dedicated equipment bay, 3-phase power, chiller provision — see RFQ-1000-HT-001), Eclipse remained the selection and all subsequent engineering (RFQ-1000-HT-011) was built around it.
