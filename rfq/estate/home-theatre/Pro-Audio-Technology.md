# RFQ-1000-HT-005 — Pro Audio Technology Speaker Array

## Overview

Pro Audio Technology (PAT) speakers form the complete loudspeaker array for the Meridian Estate theatre. This is a reference-grade, commercial-quality installation. All speakers are driven by Trinnov Altitude32 processing and Trinnov Amplitude16 amplification, with supplementary PAT amplification for the subwoofer array.

---

## 1. Speaker Configuration

| Role | Model | Quantity |
|---|---|---|
| Main array (front + surround) | S-2215sm | 11 |
| Height / Atmos overhead | S-2215sm | 4 |
| Height / Atmos surround | SCRS-25ica | 6 |
| Subwoofers | LFC-24sm | 11 |
| **Total units** | | **32** |

## 2. Configuration Notes

- **S-2215sm (main 11×):** Front L/C/R, side surround, rear surround layers
- **S-2215sm (overhead 4×):** Overhead Atmos effects channels
- **SCRS-25ica (6×):** In-ceiling height surround channels
- **LFC-24sm (11×):** Distributed subwoofer array for even bass distribution — eliminates bass nulls in large room; provides reference-level infrasonic capability

## 3. Amplification

- Primary: Trinnov Amplitude16 (16 channels from Altitude32)
- Supplementary: Pro Audio Technology amplifiers for remaining channels and subwoofer array
- All amplification housed in adjacent equipment bay rack

## 4. Calibration

- Full Trinnov 3D calibration measurement using Altitude32 microphone
- Speaker positions, distances, and levels mapped in Trinnov's 3D acoustic model
- Fine-tuning per PAT manufacturer recommendations
- Post-calibration listening session with Kevin required before sign-off

## 5. Room Integration

- All speakers installed during acoustic shell construction
- Installation positions to be confirmed with acoustic designer and PAT installer
- Speaker baffles and grilles to match theatre aesthetic (dark finish, flush with acoustic panels)
- Bass trap positioning to work alongside 11× LFC-24sm subwoofer array
