# RFQ-1000-HT-010 — Magnetar UDP900MKII Disc Transport

## Decision

Magnetar UDP900MKII confirmed and **locked** as the sole physical disc transport for the Meridian Estate theatre — the UDP800MKII alternative tier was considered and explicitly rejected in favour of locking to the single flagship model.

## Rationale

Magnetar complements rather than replaces Kaleidescape (RFQ-1000-HT-003): Kaleidescape handles server playback and file-based content at the highest quality, while Magnetar provides the ultimate native physical disc playback (UHD Blu-ray, SACD, DVD-Audio) for users who value owning/collecting discs.

## Specification

| Parameter | Specification |
|---|---|
| Model | Magnetar UDP900MKII |
| Format support | 4K UHD Blu-ray, SACD, CD, DVD-Audio |
| Outputs | Dual HDMI (dedicated audio + video paths) for optimal Trinnov integration |
| DAC | Premium internal DAC section; balanced XLR + unbalanced RCA analogue outputs |
| HDR | HDR10, HDR10+, Dolby Vision passthrough |
| Audio formats | Dolby TrueHD, DTS-HD MA, DSD, PCM (full-resolution, no downmixing) |
| Local supply | Audio Connection, Carlton Audio Visual, Instyle HiFi (AU dealers) |

## Integration

- Rack-mounted with vibration isolation and active cooling/ventilation in the equipment bay (same bay as Christie Eclipse electronics, see RFQ-1000-HT-001)
- HDMI 2.1 cabling (48Gbps AOC where distance requires)
- Installer to configure EDID, HDR passthrough, and Trinnov Altitude32 HDMI input assignment (dedicated audio HDMI input required — confirm with Trinnov integrator per RFQ-1000-HT-002)
