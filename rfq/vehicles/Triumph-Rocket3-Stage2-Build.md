# RFQ-1009-002 — Triumph Rocket 3 Stage-2 Supercharged Build

## Base Vehicle

Triumph Rocket 3 GT/Storm (2025 MY), ride-away ~AUD $38,000.

## 1. Supercharger — Forced Induction

**Selected:** Rotrex-based kit from TTS Performance, Stage-2 package — the proven centrifugal supercharger conversion for the Rocket 3, including dedicated intercooler, race-level radiator, and tailored ECU mapping path.

| Item | Specification | Cost (AUD) |
|---|---|---|
| Supercharger kit | TTS/Rotrex Stage-2 (incl. intercooler, brackets, hoses, injectors, clutch springs) | ~$10,679 landed |
| Import/shipping | 20% on kit | ~$2,136 |

**Mandatory supporting upgrades:** full ECU management/dyno tune specific to boost level; upgraded fuel system (pump + injectors); cooling upgrade (aluminium radiator + bespoke intercooler, included in Stage-2); clutch/transmission reinforcement; engine clearance/sump/breather upgrades per TTS spec.

| Item | Specification | Cost (AUD) |

## 6. Connectivity — My Triumph Connectivity System

- Triumph Bluetooth Connectivity Module (part A9820200) — TFT-displayed turn-by-turn arrows, caller ID, SMS preview, media control
- **Limitation:** does not support Siri voice for spoken navigation; uses My Triumph app text-to-speech instead
- Under-seat wireless charging pod (waterproof) combinable with OEM Bluetooth via Quad Lock MAG system
